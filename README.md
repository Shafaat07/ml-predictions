# Machine Learning Prediction Suite

This repository contains multiple machine learning projects focused on prediction and classification tasks.

## Projects

### 1. Loan Prediction
Predicts loan approval using classification models.

### 2. House Price Prediction
Regression model to estimate house prices.

### 3. Classification Models
General ML classification experiments.

## Tech Stack
- Python
- Pandas
- scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing and cleaning
- Feature engineering
- Model training and evaluation
- Comparison of multiple ML algorithms

## Results
- Loan Prediction: ~85% accuracy
- House Price Prediction: Regression model with strong predictive performance

## Purpose
Demonstrates building end-to-end ML pipelines from raw data to evaluation.
